## CRONTAB
### Commands
+ `/etc/rc.d/cron restart` -- restart service
+ `crontab -l` -- print the entries from crontab
+ `crontab -e` -- create and edit a user crontab entries
+ `crontab -r` -- remove all of the cron jobs in a user 

### Files
+ `/var/log/cron` -- logs

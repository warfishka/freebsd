## CRONTAB
### Комманды
+ `/etc/rc.d/cron restart` -- перезагрузить сервис
+ `crontab -l` -- показать все
+ `crontab -e` -- создать крон файл
+ `crontab -r` -- удалить все

### Файлы
+ `/var/log/cron` -- логи
+ `/var/cron/tabs` -- пользовательнские файлы
+ `/etc/crontab` -- рутовский файл
